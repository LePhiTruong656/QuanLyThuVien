# QuanLyThuVien
Project Quản Lý Thư Viện (LIBRARY MANAGEMENT SYSTEM) - Hệ thống quản lý thư viện được xây dựng nhằm tự động hóa các nghiệp vụ thư viện bao gồm: quản lý độc giả, quản lý sách, xử lý mượn/trả sách, tính tiền phạt và thống kê báo cáo. Hệ thống cho phép thay đổi linh hoạt các quy định mượn sách theo nhu cầu thực tế.
